# Bioinformatics Sequence Analyzer

A **Python GUI application** for analyzing DNA sequences using multiple bioinformatics algorithms. Users can import FASTA files, select sequences, run algorithms, and save a history of operations.  

This tool is perfect for students, researchers, and bioinformatics enthusiasts.

---

## Features

- Import multiple sequences from **FASTA files**.
- Apply algorithms on sequences:
  - **GC Percentage**
  - **Reverse Sequence**
  - **Complement**
  - **Reverse Complement**
  - **DNA Translation**
  - **Suffix Array**
  - **Inverse Suffix Array**
  - **Boyerâ€“Moore Pattern Search**
  - **Overlap Graph**
  - **Hamming Distance**
  - **Edit Distance**
- **History Tracking** of all operations.
- **Save History** to a `.txt` file with sequence name, operation, and result.
- **Clear History** with confirmation prompt.
